AMD Chelsea

Chelsea
Block Diagram
CU Diagram
LDS Diagram
AMD's Chelsea GPU uses the GCN 1.0 architecture and is made using a 28 nm production process at TSMC. With a die size of 123 mm² and a transistor count of 1,500 million it is a small chip. Chelsea supports DirectX 12 (Feature Level 11_1). For GPU compute applications, OpenCL version 2.1 (1.2) can be used. It features 512 shading units, 32 texture mapping units and 16 ROPs.
Further reading: GCN 1.0 Architecture Whitepaper , Southern Islands Instruction Set Architecture
